# WayOS-1.0
Shitty mini-OS made by 3 teenagers in python and A LOT of free time. Hope u enjoy &lt;3
It's free, open source and full of soul! hope u enjoy tinkering with our code!!1!! :D
Features: 
-Terminal with commands (motd, joke, date..)
- Mini file system (Mini nautilus)
- Simple games but with style (Tic-Tac-Toe, Snake, etc! :D)
### How to run?
` git clone https://github.com/pocofan1264/WayOS-1.0.git && cd WayOS-1.0 && python WayOSV1.0.py
